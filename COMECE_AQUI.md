## ⚡ INÍCIO RÁPIDO (5 passos)

### 1️⃣ Instalar Git (se não tiver)
https://git-scm.com/downloads

### 2️⃣ Abrir terminal nesta pasta e fazer:
```bash
git init
git add .
git commit -m "LUMEOR Beauty - Pronto para Vercel"
```

### 3️⃣ Criar repo no GitHub
1. Acesse: https://github.com/new
2. Nome: `lumeor-beauty`
3. Clique em "Create repository"
4. Copie o link

### 4️⃣ Fazer push do código
```bash
git remote add origin COLE_AQUI_O_LINK_DO_GITHUB
git branch -M main
git push -u origin main
```

### 5️⃣ Deploy no Vercel
1. Acesse: https://vercel.com/new
2. Clique em "Import Git Repository"
3. Conecte sua conta GitHub
4. Selecione `lumeor-beauty`
5. Clique em **DEPLOY** 🎉
